# InstaScience - Instagram Bilgi Analiz Aracı

InstaScience, kullanıcıların Instagram hesaplarının takipçi sayıları, takip ettikleri kişi sayıları ve paylaştıkları gönderi sayıları gibi verilerini çekmek için geliştirilmiş bir Python aracıdır. Program, BeautifulSoup ve Tkinter gibi kütüphaneler kullanılarak oluşturulmuştur.

## Kullanım

1. Python yüklü değilse indirip yükleyin.
2. Gerekli kütüphaneleri yüklemek için terminal veya komut istemcisinde şu komutu çalıştırın:

   ```
    pip install beautifulsoup4
 
   ```

3. Tkinter kütüphanesi genellikle Python ile birlikte gelir, ancak yine de yüklü değilse, aynı şekilde yükleyin.
4. Repoyu klonlayın veya dosyaları indirin.
5. `instascience.py` dosyasını bir Python ortamında çalıştırın.
6. Açılan pencerede bir Instagram kullanıcı adı girin.
7. "Analiz Et" butonuna tıklayın.
8. Program, belirtilen hesabın takipçi sayısı, takip edilen sayısı ve gönderi sayısını ekranda gösterecektir.

## Geliştirme Ortamı

- Python
- BeautifulSoup
- Tkinter

## Notlar

- InstaScience, Instagram'ın resmi API'sini kullanmadan, web scraping yöntemiyle veri çeker. Bu nedenle, Instagram'ın sayfa yapısındaki değişikliklerden etkilenebilir.
- Kullanıcı adı girilirken, herhangi bir boşluk veya özel karakter kullanmayın.

Bu README dosyası, InstaScience'in nasıl kullanılacağı ve geliştirileceği hakkında bilgi sağlamaktadır.
